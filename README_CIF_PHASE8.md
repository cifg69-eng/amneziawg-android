# Cif VPN Phase 8 — Exact Product UI

Этот пакет переносит в приложение структуру утверждённого левого макета:

- верхняя шапка Cif VPN;
- карточка состояния защиты;
- большая объёмная круглая кнопка;
- анимированный световой бегунок по окружности;
- таймер текущей сессии;
- карточка текущего профиля;
- три действия: белый список, импорт, настройки;
- реальная скорость соединения по изменению RX + TX статистики;
- выбор цвета интерфейса прямо из приложения;
- launcher icon «Вихрь», вариант 4, без белой рамки.

## Реальная логика

- Кнопка вызывает существующий `BaseFragment.setTunnelState`.
- Скорость рассчитывается по реальным байтам `Statistics.totalRx()` + `totalTx()`.
- Белый список сохраняет package names в `ExcludedApplications`.
- Backend проекта уже применяет их через `VpnService.Builder.addDisallowedApplication`.
- Цвет хранится локально в `SharedPreferences` и меняется без вмешательства в VPN engine.

## Установка

Распаковать содержимое в корень репозитория с заменой файлов.

Затем:

1. GitHub Desktop → Commit: `Cif VPN phase 8 exact product UI`
2. Push origin
3. GitHub Actions
4. Проверить `./gradlew :ui:assembleDebug`

Важное ограничение: показано реальное имя профиля, а не выдуманная геолокация сервера. Это сделано намеренно, чтобы интерфейс не сообщал ложные данные.
