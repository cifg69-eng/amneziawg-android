Cif VPN — финальный overlay-пакет

Что входит:
1. Чистый рабочий UI без фейковой VPN-кнопки.
2. Название приложения: Cif VPN.
3. Новая иконка приложения Cif VPN во всех Android density-папках.
4. Реальная логика туннеля остается штатной: список туннелей + настоящий переключатель справа.

Как применить:
1. Распаковать архив в корень проекта:
   C:\Users\kufik\Documents\GitHub\amneziawg-android
2. Согласиться на замену файлов.
3. Открыть GitHub Desktop.
4. Commit message: Cif VPN final icon and clean UI
5. Нажать Commit to master.
6. Нажать Push origin.
7. Открыть GitHub Actions и дождаться новой сборки APK.

Важно:
Если на телефоне останется старая иконка, удали старое приложение и поставь новый APK заново. Некоторые лаунчеры Android кэшируют иконки.
